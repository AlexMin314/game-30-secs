# game-every-30-secs
Project 1


## Stack
* Javascript, HTML5(no canvas), CSS3


## How to play
* Dodge the dots.
* Capture the stars.
* Keep yourself away from the laser.

* Move your red dot using a mouse.
* Pause the game with spacebar. (3 times limit)


## Future update
* Shild skill (Q key)
* Slower skill (W key)
